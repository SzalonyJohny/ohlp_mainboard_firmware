Constructor for option type. For example:
option f1("-c", set_current);
Make sure that you compiler support template argument deduction.

Precondition
argument life time is longer than object itself.
Parameters
[in]argumentstring literal! eg. "-m". Object stores only pointer to argument.
[in]functorfunction to be executed if command is correct. Function must have const char * as argument.
If command is matched parser will pass pointer to value after command.
Lambda type functions are suported.
